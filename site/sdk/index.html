Python SDK

The LangStitch SDK —
code your LangGraph apps cleanly

Python decorators, typed config, and a CLI for building LangGraph applications with a clean, exportable project scaffold — the same structure LangStitch and LangTailor generate from the canvas.

Install the SDK View on PyPI GitHub
Python 3.10+ LangGraph LangChain Typed MIT

Install

pip install langstitch

Install the base package, or pull in optional extras for graph, LLM, server, and HTTP integrations.

Base install

pip install langstitch

With extras

pip install "langstitch[graph]"   # langgraph + langchain-core
pip install "langstitch[llm]"     # langchain
pip install "langstitch[server]"  # fastapi + uvicorn
pip install "langstitch[all]"     # everything

What the SDK gives you

◇

Decorators

Declare nodes, tools, and agents with lightweight decorators that map straight onto a LangGraph StateGraph.

⚙️

Typed config

Strongly-typed configuration with YAML loading, so secrets resolve from the environment — never inlined.

⌨️

CLI

The langstitch command scaffolds a clean multi-module project and runs your graph locally.

🔁

Canvas round-trip

The same scaffold LangStitch and LangTailor export — design visually, then drop into code with no rewrite.

🧩

Composable extras

Opt into graph, llm, server, and http extras only when you need them.

🔓

Open source

MIT-licensed and typed end to end. Self-host friendly, no lock-in.

Quickstart

Scaffold a project, then run it. The generated layout matches a LangStitch Python export, so canvas and code stay in sync. 

pip install "langstitch[all]"

# scaffold a new project
langstitch init my-agent
cd my-agent

# run the graph
python -m my_agent
