Get started in 1–2 steps

0 steps — try now

Use the live canvas above or open the full IDE.

Open IDE →

2 steps — Docker

git clone https://github.com/vijayptiwari/LangStitch.git
cd LangStitch
docker compose -f docker-compose.prod.yml up

Open http://localhost:8080

2 steps — local dev

npm install && pip install -r server/requirements.txt
npm start

Open http://localhost:5173
