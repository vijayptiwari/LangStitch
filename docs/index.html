Visual LangGraph IDE

LangStitch

Design agent workflows on a canvas, connect tools and sub-agents, export Python, and deploy with the platform API.

Open LangStitch App View on GitHub

Features

Visual canvas

Drag-and-drop nodes: LLM, Tool, Agent, Decision, Subgraph, Function, Start/End.

Integrations

Tool registry, agent registry, MCP Studio, remote graphs, A2A, LangSmith & Langfuse.

Codegen

Live Python StateGraph export with lifecycle hooks, checkpointers, and logging.

Platform

Git sync, export/import, snapshots, Docker build, and Helm deploy via FastAPI backend.

Quick start

# Clone & install
git clone https://github.com/vijayptiwari/LangStitch.git
cd LangStitch
npm install
pip install -r server/requirements.txt

# Terminal 1 — platform API
npm run dev:api

# Terminal 2 — UI
npm run dev

Open http://localhost:5173 locally, or use the hosted static app (canvas only — run the API locally for Platform features).

Basic agent

# Run smoke-test agent
npm run agent:run

# Via API (with dev:api running)
curl -X POST http://127.0.0.1:8787/api/agent/run \
  -H "Content-Type: application/json" \
  -d "{\"project_id\":\"basic_agent_test\"}"

E2E tests

npx playwright install chromium
npm run test:e2e

Playwright starts the API and Vite dev server, mimics building a basic agent in the UI, and runs the agent via API.
