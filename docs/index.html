Sub-agent, remote, A2A, or registry delegation.

Retrieval-augmented node bound to a RAG pipeline.

Multi-Intent Classifier

Special decision node — route by classified intents.

Decision / Function / Subgraph

Conditional edges, Python transforms, nested graphs.

Asset designers

Open the sidebar Assets tab to configure reusable project artifacts. All export to dedicated Python modules.

  • Skill Designer — instructions, tools, persona, prompt templates → skills/
  • Guardrails Designer — input/output policies (block, warn, redact, rewrite) → guardrails/
  • Business Rules Designer — conditions, actions, priority → rules/
  • Persona Designer — role, tone, system prompt, constraints → personas/
  • RAG Pipelines — chunking, embedding, vector / vectorless / hybrid retrieval → rag/

Python 3.13 export

Platform → Export → Python produces a buildable multi-module ZIP:

src/<package>/
  graphs/        # StateGraph definitions
  nodes/         # Per-node handlers
  skills/        # Skill modules
  prompts/       # Prompt templates
  tools/         # Tool registry
  resources/     # Static resources
  connections/   # MCP, remote, A2A
  guardrails/    # Guardrail policies
  rules/         # Business rules
  personas/      # Agent personas
  rag/           # RAG pipelines
langsmith.json   # LangSmith + LangStitch IDE metadata
langstitch.project.json
pip install -e ".[dev]"
pytest
python -m <package>

Platform panel

  • Git — init, connect remote, pull, commit & push
  • Export — Python 3.13 project, Spring gateway, or full bundle (+ Docker + Helm)
  • Import — .langstitch.json or exported ZIP
  • Versions — local snapshots with restore
  • Build / Deploy — Docker build, Helm upgrade

Workspaces: ~/.langstitch/workspaces/<project-id>/

Integrations

  • MCP Studio — stdio / SSE / streamable HTTP servers, tools, resources
  • Tool & agent registry — reusable definitions across nodes
  • Remote graphs & A2A agents
  • LangSmith & Langfuse tracing, structured JSON logging
  • Checkpointer managers — memory, Postgres, SQLite, Redis
