Visual LangGraph IDE

LangStitch

Design agent workflows on a canvas, connect tools and sub-agents, export Python, and deploy with the platform API.

Open LangStitch App View on GitHub

Run in 1–2 steps

1 step — hosted

No install. Open the canvas in your browser.

Open LangStitch →

2 steps — Docker (full stack)

git clone https://github.com/vijayptiwari/LangStitch.git
cd LangStitch
docker compose -f docker-compose.prod.yml up

Then open http://localhost:8080 (UI + Platform API).

1 click — Codespaces

On GitHub: Code → Create codespace on main. LangStitch starts automatically.

2 steps — local dev

git clone https://github.com/vijayptiwari/LangStitch.git
cd LangStitch
npm install && pip install -r server/requirements.txt
npm start

Open http://localhost:5173

Automated publishing (GitHub Actions)

  • Deploy GitHub Pages — every push to main publishes this site + the app.
  • Publish Docker — pushes ghcr.io/vijayptiwari/langstitch-web and langstitch-api.
  • CI — build, Playwright E2E, and agent smoke tests on every PR.
  • Release — tagged releases with source archives.

Features

Visual canvas

Drag-and-drop nodes: LLM, Tool, Agent, Decision, Subgraph, Function, Start/End.

Integrations

Tool registry, agent registry, MCP Studio, remote graphs, A2A, LangSmith & Langfuse.

Codegen

Live Python StateGraph export with lifecycle hooks, checkpointers, and logging.

Platform

Git sync, export/import, snapshots, Docker build, and Helm deploy via FastAPI backend.

Basic agent

npm run agent:run

curl -X POST http://127.0.0.1:8787/api/agent/run \
  -H "Content-Type: application/json" \
  -d "{\"project_id\":\"basic_agent_test\"}"

E2E tests

npx playwright install chromium
